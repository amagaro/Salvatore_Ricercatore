# 💡 Concetto: Target User Profile (Profilo dell'utente target)

## Definizione
Il **Target User Profile** è la categorizzazione scientifica delle esigenze, delle vulnerabilità e delle capacità residue (motorie, sensoriali, cognitive) degli utenti che utilizzeranno un giardino terapeutico. Una progettazione efficace deve abbandonare il "verde generico" per strutturare lo spazio in base a profili specifici (es. bambini autistici, anziani con Alzheimer, pazienti oncologici), massimizzando l'impatto clinico dell'ambiente.
